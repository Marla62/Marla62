# Koa是什么

## 基本概念

Koa是一个新的web框架, 由express原班人马打造.

致力于成为 **web应用和API开发** 领域中的一个**更小,更富有表现力,更健壮**的基石.

## 特性

利用async函数,丢弃旧的回调函数,并有力的增强错误处理.

Koa没有捆绑任何中间件,提供了一套优雅的方法,帮助快速而愉快地编写服务端应用程序.
